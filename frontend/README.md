# Frontend

Run these commands from the `frontend/` directory.

## Install dependencies
```bash
npm install
```

## Start the dev server
```bash
npm run dev
```

## Run unit tests
```bash
npm run test -- --run
```

This runs the Vitest unit tests in `tests/unit` once and exits.

## Run the integration-style regression test
```bash
npm run test:integration
```

## Lint
```bash
npm run lint
```

## Build
```bash
npm run build
```
